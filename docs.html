üéì Student Information

GET /api/name
Get the student's full name
Optional: no_cache (boolean)
GET /api/info
Get detailed student profile including grade, school, DOB, counselor, language, and cohort year
Optional: no_cache (boolean)

üìö Classes

GET /api/classes
Get list of all enrolled classes
Optional: short (boolean), six_weeks (string), no_cache (boolean)

üìä Grades

GET /api/averages
Get current grade averages for all classes
Optional: short (boolean), six_weeks (string), no_cache (boolean)
GET /api/assignments
Get detailed assignment information with grades and dates
Optional: short (boolean), six_weeks (string), no_cache (boolean)
GET /api/weightings
Get grading category weightings for each class
Optional: short (boolean), six_weeks (string), no_cache (boolean)
GET /api/gradebook
Get complete gradebook with averages, assignments, and weightings
Optional: short (boolean), six_weeks (string), no_cache (boolean)

üìã Reports

GET /api/reportcard
Get official report card with grades from all marking periods
Optional: no_cache (boolean)
GET /api/ipr
Get interim progress report showing current progress between grading periods
Optional: no_cache (boolean)
GET /api/transcript
Get full transcript with historical grades, GPA, and credits
Optional: no_cache (boolean)
GET /api/rank
Get class rank, quartile, and GPA information
Optional: no_cache (boolean)

üí° Example Usage

# Get student name
curl "http://localhost:3000/api/name?user=USERNAME&pass=PASSWORD"

# Get class averages with shortened names
curl "http://localhost:3000/api/averages?user=USERNAME&pass=PASSWORD&short=true"

# Get assignments for specific six weeks period, bypassing cache
curl "http://localhost:3000/api/assignments?user=USERNAME&pass=PASSWORD&six_weeks=3&no_cache=true"

# Get complete gradebook
curl "http://localhost:3000/api/gradebook?user=USERNAME&pass=PASSWORD"
